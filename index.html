Tool Info

Swapping like crazy? Having a hard time combing through all those transactions to figure out how much you made? This service will analyze the transaction history of a user provided Algorand Wallet address. This includes all transactions such as liquidity providing on Tinyman, swaps between any two ASAs, buying/selling/listing/delisting NFTs, and direct transfers (e.g. rewards received from AlgoStake). This data can then be used to determine things like your cost basis and capital gain/loss from swapping ASAs and flipping NFTs.

Transaction Data

This service will provide you with your compiled transaction history in two forms; csvs and excel workbooks. One csv for each swap pair (e.g. ALGO/STKE, ALGO/DEGEN), NFT transaction (AGA197/ALGO), and single transactions (e.g. receiving claimed staking rewards from algostake for BIRDS, directly send or receiving ALGO or any ASA). The excel workbooks will be contain swaps, NFTs, and single transactions separately. Each workbook will contain a separate spreadsheet for the each pair (see example images below). The main advantage of the excel worksbooks is that they can be viewed in a GUI like Microsoft Excel (paid) or Google Sheets (free).
